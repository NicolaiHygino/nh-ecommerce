# nh-ecommerce

nh-ecommerce é um mini e-commerce fullstack construído para exercitar, de ponta a ponta, um fluxo de checkout real: carrinho, criação de pedido, pagamento integrado via webhook (modo sandbox) e atualização atômica de estoque com rollback em caso de falha. Backend em Spring Boot 3 / Java 17, com autenticação JWT, controle de acesso por dono do recurso e testes de integração com Testcontainers. Frontend em Vite + React 19 + TypeScript, com TanStack Query e shadcn/ui.
⚠️ Projeto de portfólio — pagamentos rodam em modo sandbox, sem transação real.
